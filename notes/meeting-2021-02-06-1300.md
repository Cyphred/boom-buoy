## Device Specifications
- 3D printed enclosure.
- Primary function is monitoring for explosives.
- No triangulation, per-node detection.
- Simple LED dashboard with indicator lights. If a node detects an explosion, its corresponding LED will be lit, accompanied by a buzzer.
- RF transceivers for wireless communication. These typically have a 1km range in unobstructed space.
- Each node to be powered by an Arduino nano.
- Power will be supplied via batteries charged by solar panels on-board.
